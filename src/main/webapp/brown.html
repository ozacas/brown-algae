About

The major aim of this study was to identify arabinogalactan proteins (AGPs) and related proteins in Brown algae. Specifically, Ectocarpus . We briefly explored other brown algae for which sequence data is available, however, this work is ongoing.

Pipeline

To meet the aims of the study, we constructed a simplistic pipeline to identify the major features of these molecules:
  • N-terminal signal peptide, as predicted by SignalP
  • Glycomotifs which are proline-rich (often hydroxyproline) and glycosylated, post-translationally
  • a GPI anchor, as predicted with BigPI
This was applied to the available proteome sequence data as downloaded from the Ectocarpus siliculosus genome project.

Limitations

The performance of computational predictors is not well established for algae, we assume that the performance of SignalP and Big-PI is comparable to performance for higher-plants. The aim of this pipeline was to generate a modest list of candidates, suitable for experimental validation, rather than obtaining the highest possible precision. Re-running the pipeline is only possible if you have access to the University of Melbourne network, as access to the SignalP server is access-restricted.

Installing the pipeline

Installation of the pipeline requires you to carry out three steps:
  • Install KNIME via knime.org (v2.11 or later)
  • Install KNIME PlantCell extension
  • Download and import the workflow below into your KNIME workspace (file->import). Do not unpack the zip file, KNIME will do that.
